# Página de Download Seguro (GitHub Pages)

Este repositório já está pronto para ser enviado ao GitHub.

## COMO USAR:

1. Crie um repositório no GitHub.
2. Envie este arquivo `index.html` junto com o `README.md`.
3. Vá em **Settings → Pages**.
4. Em "Source", selecione **main branch**.
5. A página será publicada em:
   https://seuusuario.github.io/seurepositorio/

## COMO TROCAR O ARQUIVO DE DOWNLOAD:

Abra o arquivo `index.html` e altere esta linha:

```
const linkArquivo = "COLE_SEU_LINK_AQUI";
```

Cole ali o novo link direto do Archive.org.

A página continua a mesma, mas o download muda automaticamente.
